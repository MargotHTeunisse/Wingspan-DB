<h1>What have I landed on?</h1>

This is a personal project revolving around the popular bird-themed board game Wingspan. I enjoy the game very much, and had the idea that it could be the basis for a nice little data analysis project. 

To test the relational database structure, I have started with only the ten birds from the Swift Start pack. Building on this, I plan to add other birds, so that I can do some statistical analysis of their varying properties.
